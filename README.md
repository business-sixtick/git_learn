# git_learn
깃, 깃허브를 학습. 학습내용을 정리

- git clone git@github.com:business-sixtick/git_learn.git
- cd git_learn
- flutter create .
- flutter pub get
- flutter pub add flutter_riverpod
- flutter pub add window_size
- flutter pub add url_launcher

# 배포 
- flutter doctor
- flutter build web
- 웹 테스트 flutter run -d chrome
